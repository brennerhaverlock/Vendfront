## Installation

Clone Repo
```
git clone https://github.com/brennerhaverlock/Vendfront.git
```

### Install dependencies

```
npm install 
```

### Create .env file in root directory with these parameters 

```
touch .env
```

```
MONGODB_URI=mongodb://localhost/<yourprojectname>
```

### Deployment to local

``` 
npm run dev
```
